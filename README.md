# 📊 RetailHub Kenya Sales Performance Dashboard


## Project Overview
Interactive Excel dashboard built to analyze retail sales performance using KPI cards, PivotTables, PivotCharts and slicers.

## Business Problem
Provide management with a single dashboard for monitoring sales, profits, products, regions and salesperson performance.

## Dashboard Features
- KPI Cards
- Interactive Slicers
- Sales by Region
- Monthly Sales Trend
- Sales by Category
- Top Products
- Salesperson Performance
- Profit by Region

## Tools
- Microsoft Excel
- PivotTables
- PivotCharts
- Slicers
- Conditional Formatting

## Key Insights
- Nakuru recorded the highest sales and profit.
- Electronics generated the largest share of revenue.
- Laptops were the top-performing product.
- March showed the strongest monthly sales.
- Mombasa recorded the lowest profit.

## Recommendations
- Increase investment in high-performing regions.
- Improve low-performing regions.
- Monitor KPIs regularly.
